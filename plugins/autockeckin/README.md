# 站点自动签到插件

自动签到 MoviePilot 中配置的所有 PT 站点。

## 功能特性

- ✅ 支持 NexusPHP、Gazelle 等主流站点框架
- ✅ 自动检测签到页面（attendance.php 等）
- ✅ 支持排除特定站点（如需要验证码的站点）
- ✅ 支持仅签到指定站点
- ✅ 签到结果通知推送
- ✅ Cron 定时执行
- ✅ API 手动触发

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用插件 | 开启/关闭自动签到 | 关闭 |
| 开启通知 | 签到完成后发送通知 | 开启 |
| 签到周期 | Cron 表达式 | `0 8 * * *`（每天8点） |
| 排除站点 | 逗号分隔的站点名称 | 空 |
| 仅签到站点 | 仅签到指定站点（留空签所有） | 空 |
| 立即运行一次 | 立即执行一次签到 | 关闭 |

## API

- `GET /api/v1/autockeckin/checkin?apikey=xxx` - 手动触发签到

## 支持的签到方式

1. **NexusPHP attendance.php** - 最常见的签到页面
2. **签到表单提交** - 部分站点的签到按钮
3. **签到 API** - index.php?action=addbonus 等

## 注意事项

- 对于需要验证码的站点（如北洋园），建议添加到排除列表
- 签到间隔为 2 秒，避免请求过快被站点封禁
- Cookie 失效时会跳过并通知
